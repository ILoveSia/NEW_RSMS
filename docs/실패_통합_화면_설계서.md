## 🎨 개요

### 시스템 정보 (RSMS 아키텍처 표준)
- **시스템명**: 차세대 통합 책무구조도 이행관리시스템 (Next-Gen RSMS)
- **프론트엔드 스택**: React 18.3.1 + TypeScript 5.5.2 + Vite 5.3.1
- **UI 컴포넌트**: Material-UI 5.16.0 + AG-Grid 32.0.0 + Recharts 2.12.7
- **스타일링**: CSS Modules + SCSS (인라인 스타일 금지)
- **상태 관리**: Zustand 4.5.2 + TanStack Query 5.45.1
- **아키텍처**: Domain-Driven Design + Atomic Design Pattern
- **다국어**: React i18next 13.5.0
- **작성일**: 2025-09-12
- **문서 버전**: 2.0 (RSMS 아키텍처 적용)
- **설계 철학**: "RSMS 아키텍처 기반의 확장 가능한 컴포넌트 시스템"

### RSMS 아키텍처 기반 설계 원칙
- **컴포넌트 재사용**: shared/components 기반의 공통 컴포넌트 활용
- **타입 안전성**: TypeScript + React.FC<Props> 엄격한 타입 정의 (any 타입 금지)
- **스타일 일관성**: CSS Modules로 스타일 충돌 방지
- **접근성**: WCAG 2.1 AA 준수 + 한국 웹 접근성 지침 준수
- **성능 최적화**: React.memo, useMemo, useCallback 활용
- **반응형 디자인**: SCSS breakpoints와 Material-UI Grid 시스템

### RSMS 아키텍처 기반 통합 설계 특징
- **Atomic Design Pattern**: atoms/molecules/organisms/templates 계층 구조
- **Domain-Driven Design**: 도메인별 컴포넌트 모듈화 (auth, users, resps, reports, dashboard, settings)
- **Material-UI 5.16 표준**: 공통 컴포넌트 시스템 (Button, SearchBar, BaseDataGrid)
- **CSS Modules + SCSS**: 완전한 스타일 캐슡화 (인라인 스타일 절대 금지)
- **검증된 안정성**: my_rsms ITCEN의 NULL 안전 패턴 + RSMS 타입 안전성
- **실시간 데이터**: TanStack Query와 Zustand를 활용한 상태 관리
- **다국어 지원**: React i18next로 한국어/영어 지원

```typescript
// shared/components/organisms/BaseDataGrid/BaseDataGrid.tsx
import React from 'react';
import { AgGridReact } from 'ag-grid-react';
import { ColDef } from 'ag-grid-community';
import 'ag-grid-community/styles/ag-grid.css';
import 'ag-grid-community/styles/ag-theme-material.css';
import styles from './BaseDataGrid.module.scss';

interface BaseDataGridProps<T> {
  data: T[];
  columns: ColDef<T>[];
  loading?: boolean;
  pagination?: boolean;
  pageSize?: number;
  onRowClick?: (row: T) => void;
  height?: string | number;

interface DataGridProps<T> {
  data: T[];
  columns: DataGridColumn<T>[];
  loading?: boolean;
  pagination?: boolean;
  pageSize?: number;
  selection?: 'single' | 'multiple' | 'none';
  onRowSelect?: (rows: T[]) => void;
  onRowDoubleClick?: (row: T) => void;
  height?: number;
}

// RSMS 표준: BaseDataGrid 직접 AgGridReact 사용 금지
export function BaseDataGrid<T>({
  data,
  columns,
  loading = false,
  pagination = true,
  pageSize = 20,
  onRowClick,
  height = 400
}: DataGridProps<T>) => {
  const [selectedRows, setSelectedRows] = useState<T[]>([]);
  
  const columnDefs = useMemo(() => 
    columns.map(col => ({
      field: col.field as string,
      headerName: col.headerName,
      width: col.width || 150,
      sortable: col.sortable !== false,
      filter: col.filterable !== false,
      cellRenderer: col.cellRenderer,
      checkboxSelection: col.field === 'id' && selection !== 'none'
    })), [columns, selection]
  );

  return (
    <Paper elevation={1} sx={{ height: height + 60 }}>
      <Box p={2} display="flex" justifyContent="space-between" alignItems="center">
        <Typography variant="h6">
          총 {data.length.toLocaleString()}건
        </Typography>
        
        <Box display="flex" gap={1}>
          <IconButton size="small" title="새로고침">
            <RefreshIcon />
          </IconButton>
          <IconButton size="small" title="Excel 다운로드">
            <FileDownloadIcon />
          </IconButton>
          <IconButton size="small" title="컬럼 설정">
            <SettingsIcon />
          </IconButton>
        </Box>
      </Box>
      
      <Box className="ag-theme-material" style={{ height, width: '100%' }}>
        <AgGridReact
          rowData={data}
          columnDefs={columnDefs}
          pagination={pagination}
          paginationPageSize={pageSize}
          rowSelection={selection}
          onSelectionChanged={(params) => {
            const selected = params.api.getSelectedRows();
            setSelectedRows(selected);
            onRowSelect?.(selected);
          }}
          onRowDoubleClicked={(params) => {
            onRowDoubleClick?.(params.data);
          }}
          loadingOverlayComponent="로딩 중..."
          noRowsOverlayComponent="데이터가 없습니다."
        />
      </Box>
    </Paper>
  );
};
```

#### Chart Component (Recharts 래핑)
```typescript
interface ChartProps {
  type: 'line' | 'bar' | 'pie' | 'area' | 'scatter';
  data: any[];
  xField: string;
  yField: string | string[];
  title?: string;
  height?: number;
  colors?: string[];
  showLegend?: boolean;
  showGrid?: boolean;
  showTooltip?: boolean;
}

const Chart: React.FC<ChartProps> = ({
  type,
  data,
  xField,
  yField,
  title,
  height = 300,
  colors = ['#2196f3', '#4caf50', '#ff9800', '#f44336'],
  showLegend = true,
  showGrid = true,
  showTooltip = true
}) => {
  const renderChart = () => {
    const commonProps = {
      data,
      width: '100%',
      height: height - 60
    };

    switch (type) {
      case 'line':
        return (
          <LineChart {...commonProps}>
            {showGrid && <CartesianGrid strokeDasharray="3 3" />}
            <XAxis dataKey={xField} />
            <YAxis />
            {showTooltip && <Tooltip />}
            {showLegend && <Legend />}
            {Array.isArray(yField) ? 
              yField.map((field, index) => (
                <Line 
                  key={field}
                  type="monotone" 
                  dataKey={field} 
                  stroke={colors[index % colors.length]} 
                  strokeWidth={2}
                />
              )) :
              <Line 
                type="monotone" 
                dataKey={yField} 
                stroke={colors[0]} 
                strokeWidth={2}
              />
            }
          </LineChart>
        );
        
      case 'bar':
        return (
          <BarChart {...commonProps}>
            {showGrid && <CartesianGrid strokeDasharray="3 3" />}
            <XAxis dataKey={xField} />
            <YAxis />
            {showTooltip && <Tooltip />}
            {showLegend && <Legend />}
            {Array.isArray(yField) ? 
              yField.map((field, index) => (
                <Bar 
                  key={field}
                  dataKey={field} 
                  fill={colors[index % colors.length]}
                />
              )) :
              <Bar 
                dataKey={yField} 
                fill={colors[0]}
              />
            }
          </BarChart>
        );
        
      // ... 다른 차트 타입들
    }
  };

  return (
    <Paper elevation={1} sx={{ p: 2, height }}>
      {title && (
        <Typography variant="h6" gutterBottom>
          {title}
        </Typography>
      )}
      
      <ResponsiveContainer>
        {renderChart()}
      </ResponsiveContainer>
    </Paper>
  );
};
```

```typescript
            ...field.validation
          }}
          render={({ field: controllerField }) => {
            switch (field.type) {
              case 'text':
              case 'email':
                return (
                  <TextField
                    {...controllerField}
                    label={field.label}
                    placeholder={field.placeholder}
                    type={field.type}
                    fullWidth
                    required={field.required}
                    error={!!errors[field.name]}
                    helperText={errors[field.name]?.message}
                    variant="outlined"
                  />
                );
                
              case 'number':
                return (
                  <TextField
                    {...controllerField}
                    label={field.label}
                    placeholder={field.placeholder}
                    type="number"
                    fullWidth
                    required={field.required}
                    error={!!errors[field.name]}
                    helperText={errors[field.name]?.message}
                    variant="outlined"
                    inputProps={{
                      min: field.validation?.min,
                      max: field.validation?.max
                    }}
                  />
                );
                
              case 'select':
                return (
                  <FormControl fullWidth required={field.required} error={!!errors[field.name]}>
                    <InputLabel>{field.label}</InputLabel>
                    <Select
                      {...controllerField}
                      label={field.label}
                    >
                      {field.options?.map(option => (
                        <MenuItem key={option.value} value={option.value}>
                          {option.label}
                        </MenuItem>
                      ))}
                    </Select>
                    {errors[field.name] && (
                      <FormHelperText>{errors[field.name]?.message}</FormHelperText>
                    )}
                  </FormControl>
                );
                
              case 'textarea':
                return (
                  <TextField
                    {...controllerField}
                    label={field.label}
                    placeholder={field.placeholder}
                    fullWidth
                    multiline
                    rows={4}
                    required={field.required}
                    error={!!errors[field.name]}
                    helperText={errors[field.name]?.message}
                    variant="outlined"
                  />
                );
                
              case 'date':
                return (
                  <LocalizationProvider dateAdapter={AdapterDayjs}>
                    <DatePicker
                      label={field.label}
                      value={controllerField.value}
                      onChange={controllerField.onChange}
                      renderInput={(params) => (
                        <TextField 
                          {...params} 
                          fullWidth 
                          required={field.required}
                          error={!!errors[field.name]}
                          helperText={errors[field.name]?.message}
                        />
                      )}
                    />
                  </LocalizationProvider>
                );
                
              default:
                return null;
            }
          }}
        />
      </Grid>
    );
  };

  return (
    <Paper elevation={1} sx={{ p: 3 }}>
      <form onSubmit={handleSubmit(onSubmit)}>
        <Grid container spacing={2}>
          {fields.map(renderField)}
          
          <Grid item xs={12}>
            <Box display="flex" justifyContent="flex-end" gap={2} mt={2}>
              {onCancel && (
                <Button 
                  variant="outlined" 
                  onClick={onCancel}
                  disabled={loading}
                >
                  {cancelText}
                </Button>
              )}
              <Button 
                type="submit" 
                variant="contained" 
                disabled={loading}
                startIcon={loading ? <CircularProgress size={20} /> : undefined}
              >
                {submitText}
              </Button>
            </Box>
          </Grid>
        </Grid>
      </form>
    </Paper>
  );
};
```

```typescript
// PWA Service Worker Registration
const PWAInstallPrompt: React.FC = () => {
  const [deferredPrompt, setDeferredPrompt] = useState<any>(null);
  const [showInstallPrompt, setShowInstallPrompt] = useState(false);

  useEffect(() => {
    const handleBeforeInstallPrompt = (e: Event) => {
      e.preventDefault();
      setDeferredPrompt(e);
      setShowInstallPrompt(true);
    };

    window.addEventListener('beforeinstallprompt', handleBeforeInstallPrompt);
    
    return () => {
      window.removeEventListener('beforeinstallprompt', handleBeforeInstallPrompt);
    };
  }, []);

  const handleInstallClick = async () => {
    if (deferredPrompt) {
      deferredPrompt.prompt();
      const { outcome } = await deferredPrompt.userChoice;
      
      if (outcome === 'accepted') {
        console.log('PWA installed');
      }
      
      setDeferredPrompt(null);
      setShowInstallPrompt(false);
    }
  };

  if (!showInstallPrompt) return null;

  return (
    <Snackbar
      open={showInstallPrompt}
      onClose={() => setShowInstallPrompt(false)}
      message="앱을 홈 화면에 추가하시겠습니까?"
      action={
        <Button color="inherit" onClick={handleInstallClick}>
          설치
        </Button>
      }
    />
  );
};
```

```typescript
// Infinite Scroll Hook
const useInfiniteScroll = <T>(
  fetchMore: (page: number) => Promise<T[]>,
  initialData: T[] = []
) => {
  const [data, setData] = useState<T[]>(initialData);
  const [loading, setLoading] = useState(false);
  const [hasMore, setHasMore] = useState(true);
  const [page, setPage] = useState(1);

  const loadMore = useCallback(async () => {
    if (loading || !hasMore) return;

    setLoading(true);
    try {
      const newData = await fetchMore(page);
      
      if (newData.length === 0) {
        setHasMore(false);
      } else {
        setData(prev => [...prev, ...newData]);
        setPage(prev => prev + 1);
      }
    } catch (error) {
      console.error('Error loading more data:', error);
    } finally {
      setLoading(false);
    }
  }, [fetchMore, loading, hasMore, page]);

  // Intersection Observer for auto-loading
  const { ref, inView } = useInView({
    threshold: 0,
    rootMargin: '100px'
  });

  useEffect(() => {
    if (inView) {
      loadMore();
    }
  }, [inView, loadMore]);

  return { data, loading, hasMore, loadMore, ref };
};
```

```typescript
const AccessibleTable: React.FC<AccessibleTableProps> = ({
  data,
  columns,
  caption,
  onSort,
  sortField,
  sortDirection
}) => {
  const [focusedCell, setFocusedCell] = useState<[number, number]>([0, 0]);
  
  const handleKeyDown = (e: KeyboardEvent) => {
    const [row, col] = focusedCell;
    
    switch (e.key) {
      case 'ArrowUp':
        e.preventDefault();
        setFocusedCell([Math.max(0, row - 1), col]);
        break;
      case 'ArrowDown':
        e.preventDefault();
        setFocusedCell([Math.min(data.length - 1, row + 1), col]);
        break;
      case 'ArrowLeft':
        e.preventDefault();
        setFocusedCell([row, Math.max(0, col - 1)]);
        break;
      case 'ArrowRight':
        e.preventDefault();
        setFocusedCell([row, Math.min(columns.length - 1, col + 1)]);
        break;
      case 'Enter':
      case ' ':
        e.preventDefault();
        // Handle row selection or action
        break;

// Focus Management Hook
const useFocusManagement = () => {
  const focusElementById = (id: string) => {
    const element = document.getElementById(id);
    if (element) {
      element.focus();
    }
  };

  const focusFirstError = () => {
    const errorElement = document.querySelector('[aria-invalid="true"]') as HTMLElement;
    if (errorElement) {
      errorElement.focus();
    }
  };

  const trapFocus = (containerRef: React.RefObject<HTMLElement>) => {
    const focusableElements = containerRef.current?.querySelectorAll(
      'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
    );
    
    if (focusableElements && focusableElements.length > 0) {
      const firstElement = focusableElements[0] as HTMLElement;
      const lastElement = focusableElements[focusableElements.length - 1] as HTMLElement;
      
      const handleTabKey = (e: KeyboardEvent) => {
        if (e.key === 'Tab') {
          if (e.shiftKey) {
            if (document.activeElement === firstElement) {
              lastElement.focus();
              e.preventDefault();
            }
          } else {
            if (document.activeElement === lastElement) {
              firstElement.focus();
              e.preventDefault();
            }
          }
        }
      };
      
      containerRef.current?.addEventListener('keydown', handleTabKey);
      firstElement.focus();
      
      return () => {
        containerRef.current?.removeEventListener('keydown', handleTabKey);
      };
    }
  };

  return {
    focusElementById,
    focusFirstError,
    trapFocus
  };
};
```

### 9.2 결재함 화면 분석 및 설계

#### 9.2.1 실제 화면 구성 분석
캡처된 결재함 화면을 기반으로 한 상세 설계:

**탭 시스템**: 
- 대기 (0/10), 진행중 (1/10), 완료 (2/2), 전체 (3)
- 각 탭에 현재 건수/전체 건수 표시

**필터 영역**:
- 기간검색: 시작일 ~ 종료일 선택
- 업무 종류: 드롭다운 선택 (전체/특정 업무)  
- 부서: 텍스트 입력 (예: "0000 방기드랩")
- 조회 및 통인정보 버튼

**결재 목록 테이블**:
- 컬럼: 결재ID, 업무종류, 내용, 결재상태, 결재현황, 기안자, 요청일, 대기자
- 실제 데이터 예시: "1503, 책무구조, 관리점검, 기안, 0/1, 관리자(0000000), 2025-09-08 15:36"

#### 9.2.2 결재 상세 모달 구성
실제 결재 상세 화면 분석:

**관리업무내용 섹션**: 
- 순번, 관리업무명, 내부통제, 내부통제항목명, 수행자, 수행여부, 수행결과

**결재선 섹션**:
- 순번, 결재단계명, 구분, 결재일시, 부결, 직원번호, 직원명, 의견  
- 기안자: "기안자, 기안, 2025-09-08 15:36:06, 관리자"
- 결재자: "부장, 감사부, 0000003, FIT 3"
